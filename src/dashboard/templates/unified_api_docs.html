API Endpoints

GET /health

Check API health and availability status. Returns service status, uptime, and system information. 

curl -X GET 'http://localhost:8002/health'
GET /api/market/overview

Get comprehensive market overview including sentiment analysis, volatility metrics, trending pairs, and 6-dimensional confluence data. 

curl -X GET 'http://localhost:8002/api/market/overview'
GET /api/signals/top

Retrieve top trading signals with confluence scores, risk assessments, and actionable recommendations. 

curl -X GET 'http://localhost:8002/api/signals/top'
GET /api/dashboard/data

Access dashboard data for real-time trading interface including positions, alerts, and performance metrics. 

curl -X GET 'http://localhost:8002/api/dashboard/data'
GET /system-monitoring

View real-time system monitoring dashboard with performance metrics, resource usage, and component status. 

curl -X GET 'http://localhost:8002/system-monitoring'
GET /links

Main navigation page providing access to all system features, dashboards, and social links. 

curl -X GET 'http://localhost:8002/links'

Code Examples

JavaScript/Node.js

// Fetch market overview
const response = await fetch('http://localhost:8002/api/market/overview');
const data = await response.json();
console.log('Market sentiment:', data.sentiment);

Python

import requests

# Get trading signals
response = requests.get('http://localhost:8002/api/signals/top')
signals = response.json()
print(f'Top signal: {signals[0]["symbol"]}')

cURL

# Health check
curl -X GET 'http://localhost:8002/health' \
  -H 'Content-Type: application/json'
