BRAND VOICE GUIDELINES

DO USE:

  • • All-caps for emphasis (VIRTUOSO, SIGNALS)
  • • Technical terminology (API, WebSocket, CVD)
  • • Data-driven language (72% win rate, 6ms latency)
  • • Monospace formatting for code/data
  • • Arrow symbols for CTAs (START →)

DON'T USE:

  • • Casual language or slang
  • • Emojis (except in social media)
  • • Rounded, soft typography
  • • Bright, saturated backgrounds
  • • Generic stock imagery

DATA VISUALIZATION

Financial chart components designed for crypto trading platforms. All charts follow the Virtuoso brand style with dark cyberpunk aesthetics, neon amber accents, and IBM Plex Mono typography for data precision.

CHART ARCHITECTURE

Design Principles

  • ▸ Scannable at a glance: Critical data visible without interaction
  • ▸ Precision: IBM Plex Mono for all numeric values
  • ▸ Subtle animations: Smooth transitions, no jarring effects
  • ▸ Accessibility: High contrast, clear labels, hover tooltips

Recommended Libraries

  • 1. Lightweight Charts (TradingView): Best for candlesticks
  • 2. ApexCharts: Versatile, good TypeScript support
  • 3. Pure CSS/SVG: Simple charts, no dependencies
// Chart Color Palette
--chart-primary: #fbbf24;
--chart-secondary: #06B6D4;
--chart-success: #10b981;
--chart-error: #ef4444;
--chart-grid: rgba(255,255,255,0.03);
